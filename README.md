# 🎓 College Management System

A full-stack web application for managing college operations including students, teachers, attendance, notices, and results.

## 🚀 Features
- 🔐 AWS Cognito Authentication (Login/Signup with Email Verification)
- 👨‍🎓 Student Dashboard
- 👩‍🏫 Teacher Dashboard
- 📢 Notices System
- 📊 Attendance Management
- 📝 Results Section
- 🔄 Real-time Data Sync (Student ↔ Teacher)

## 🛠 Tech Stack
- Frontend: React / HTML / CSS / JS
- Backend: Node.js / AWS Lambda
- Database: DynamoDB / Firebase
- Authentication: AWS Cognito


## ⚙️ Setup Instructions
1. Clone repo
2. Install dependencies
3. Add `.env` file
4. Run project
